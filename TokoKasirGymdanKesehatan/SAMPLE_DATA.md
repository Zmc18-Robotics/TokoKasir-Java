## Cara Menambahkan Data

1. **Login sebagai Admin atau Owner**
   - Username: admin
   - Password: admin123

2. **Tambahkan User Baru**
   - Klik menu "Kelola User"
   - Klik "Tambah User"
   - Isi form dan pilih role
   - Klik "Simpan"

3. **Tambahkan Kategori**
   - Klik menu "Kelola Kategori"
   - Klik "Tambah Kategori"
   - Isi nama dan deskripsi
   - Klik "Simpan"

4. **Tambahkan Supplier**
   - Klik menu "Kelola Supplier"
   - Klik "Tambah Supplier"
   - Isi nama, telepon, dan alamat
   - Klik "Simpan"

5. **Tambahkan Produk**
   - Klik menu "Kelola Produk"
   - Klik "Tambah Produk"
   - Isi semua field (nama, kategori, supplier, harga, stok, diskon, deskripsi)
   - Klik "Simpan"

## Testing Transaksi

Setelah data contoh ditambahkan:

1. **Logout dan Login sebagai Kasir**
   - Username: kasir1
   - Password: kasir123

2. **Lakukan Transaksi**
   - Klik menu "Transaksi Penjualan"
   - Pilih produk dari dropdown
   - Masukkan jumlah
   - Klik "Tambah ke Keranjang"
   - Ulangi untuk produk lain
   - Klik "Cetak Struk" untuk menyelesaikan transaksi

3. **Lihat Laporan**
   - Login sebagai Owner atau Admin
   - Klik menu "Laporan Penjualan"
   - Lihat statistik dan detail transaksi

## Tips Testing

- Coba berbagai kombinasi produk
- Test diskon otomatis pada produk
- Verifikasi stok berkurang setelah transaksi
- Coba lihat detail transaksi di laporan
- Test backup database
- Coba edit dan hapus produk
- Verifikasi role-based access control

Selamat testing!
