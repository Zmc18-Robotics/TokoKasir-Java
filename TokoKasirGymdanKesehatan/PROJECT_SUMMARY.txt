================================================================================
                              PROJECT OVERVIEW
================================================================================

Aplikasi kasir toko berbasis Java dengan sistem role-based access control (RBAC)
untuk mengelola transaksi penjualan, produk, kategori, supplier, dan laporan.

Aplikasi ini dirancang untuk:
- Melayani transaksi penjualan (Kasir)
- Mengelola produk dan stok (Owner)
- Melihat laporan penjualan (Owner)
- Mengelola user dan sistem (Admin)

================================================================================
                              FITUR UTAMA
================================================================================

1. SISTEM AUTENTIKASI & OTORISASI
   - Login dengan username dan password
   - Role-based access control (KASIR, OWNER, ADMIN)
   - Session management per user
   - Default admin account

2. MODUL KASIR
   - Input transaksi penjualan
   - Pilih produk dan jumlah
   - Validasi stok otomatis
   - Perhitungan harga dengan diskon
   - Cetak struk pembelian
   - Lihat stok produk (read-only)

3. MODUL OWNER
   - CRUD Produk (Tambah, Ubah, Hapus)
   - CRUD Kategori
   - CRUD Supplier
   - Laporan penjualan dengan statistik
   - Kelola user baru
   - Backup database

4. MODUL ADMIN
   - Semua akses seperti Owner
   - Pengelolaan teknis sistem
   - Backup database

5. DATABASE & PENYIMPANAN
   - SQLite database
   - 6 tabel utama (users, categories, suppliers, products, transactions, transaction_items)
   - Foreign key constraints
   - Audit trail dengan timestamp

6. KEAMANAN
   - Validasi input
   - Unique constraints
   - Role-based access control
   - Audit trail

================================================================================
                        CARA INSTALASI & MENJALANKAN
================================================================================

1. DOWNLOAD SQLITE JDBC DRIVER
   - Download dari: https://github.com/xerial/sqlite-jdbc/releases
   - Letakkan di folder: lib/sqlite-jdbc-3.44.0.0.jar

2. COMPILE PROGRAM
   Windows:
   > javac -d bin src/*.java
   
   Linux/Mac:
   $ javac -d bin src/*.java

3. JALANKAN PROGRAM
   Windows (menggunakan batch):
   > build.bat
   
   Windows (manual):
   > java -cp bin;lib/sqlite-jdbc-3.44.0.0.jar Main
   
   Linux/Mac:
   $ java -cp bin:lib/sqlite-jdbc-3.44.0.0.jar Main

================================================================================
                          STRUKTUR DATABASE
================================================================================

TABLE: users
- id (PRIMARY KEY)
- username (UNIQUE)
- password
- role (KASIR, OWNER, ADMIN)
- created_at (TIMESTAMP)

TABLE: categories
- id (PRIMARY KEY)
- name (UNIQUE)
- description

TABLE: suppliers
- id (PRIMARY KEY)
- name (UNIQUE)
- phone
- address

TABLE: products
- id (PRIMARY KEY)
- name
- category_id (FOREIGN KEY)
- supplier_id (FOREIGN KEY)
- price
- stock
- discount
- description
- image_path
- created_at (TIMESTAMP)

TABLE: transactions
- id (PRIMARY KEY)
- user_id (FOREIGN KEY)
- total_amount
- payment_method
- created_at (TIMESTAMP)

TABLE: transaction_items
- id (PRIMARY KEY)
- transaction_id (FOREIGN KEY)
- product_id (FOREIGN KEY)
- quantity
- price
- discount
- subtotal

================================================================================
                          FITUR KEAMANAN
================================================================================

1. AUTENTIKASI
   - Login dengan username dan password
   - Validasi kredensial di database
   - Session per user

2. OTORISASI
   - Role-based access control
   - Menu dinamis sesuai role
   - Pembatasan akses fitur

3. VALIDASI INPUT
   - Field kosong tidak diterima
   - Duplikasi username/kategori/supplier ditolak
   - Validasi stok sebelum transaksi

4. AUDIT TRAIL
   - Timestamp pada setiap transaksi
   - Pencatatan user yang melakukan transaksi
   - Riwayat lengkap di laporan

5. BACKUP DATABASE
   - Backup manual dengan timestamp
   - File backup: tokokasir_backup_YYYYMMDD_HHmmss.db

================================================================================
                          PANDUAN PENGGUNAAN
================================================================================

KASIR:
1. Login dengan username kasir dan password
2. Klik "Transaksi Penjualan"
3. Pilih produk dari dropdown
4. Masukkan jumlah
5. Klik "Tambah ke Keranjang"
6. Ulangi untuk produk lain
7. Klik "Cetak Struk" untuk menyelesaikan transaksi

OWNER:
1. Login dengan username owner dan password
2. Kelola produk, kategori, supplier melalui menu
3. Lihat laporan penjualan
4. Tambah user baru
5. Backup database jika diperlukan

ADMIN:
1. Login dengan username admin dan password
2. Akses semua fitur seperti Owner
3. Kelola sistem secara keseluruhan

================================================================================
                          TROUBLESHOOTING
================================================================================

Error: "Class not found: org.sqlite.JDBC"
Solusi: Pastikan sqlite-jdbc-3.44.0.0.jar ada di folder lib/

Error: "Cannot find symbol"
Solusi: Compile ulang dengan javac -d bin src/*.java

Aplikasi tidak merespons saat startup
Solusi: Tunggu beberapa detik, database sedang diinisialisasi

Database tidak bisa ditulis
Solusi: Pastikan folder proyek memiliki write permission

Lihat SETUP.md untuk troubleshooting lebih lengkap.
